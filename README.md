# MF — Memes Finder

A mobile-focused React preview app for discovering meme templates, audio clips, and video edits.

## Features

- 🏠 **Home Screen** — Hero carousel, category grid, audio clips, best edits
- 🎭 **Categories** — Browse templates by actor/character (Sunil, Brahmi, Ali, etc.)
- 🎵 **Audio Clips** — BGMs, funny sound effects, dialog cuts
- 💬 **Chat** — Message the admin to request new template packs
- 💾 **Saved** — View saved templates, audios, trending pages & scores
- 👤 **Profile** — User profile section

## Tech Stack

- React
- Framer Motion (animations)
- Tailwind CSS (utility-first styling)

## Design

- Dark / Netflix-like theme
- Mobile-first layout (390×844 / iPhone preview)
- Glassmorphism, gradients, and micro-animations

## Getting Started

```bash
npm install
npm run dev
```
